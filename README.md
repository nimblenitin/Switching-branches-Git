# Switching-branches-Git

Steps-

```

1. On an existing local git repo create a branch
$ git branch text_branch

2. Use the following command to switch to the newly created text_branch:
$ git checkout text_branch

3. Use the following command to create a file:
$ vi index.html

4. Use the following commands to add the file to the text_branch and commit the changes:
$ git add index.html
$ git commit -a -m "file modified"

5. Check the status of the new branch using the following command:
$ git status

6. Use the following command to check the git logs:
$ git log --oneline

7. Check the current branch using the following command:
$ git branch

8. Use the following command to switch back to the main branch:
$ git checkout main

```
